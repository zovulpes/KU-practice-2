# Визуализация графа зависимостей пакетов

## Описание проекта

Этот проект реализует инструмент для анализа и визуализации зависимостей пакетов. Проект состоит из пяти этапов:

1. **Этап 1 – Минимальный прототип**

   - CLI-приложение, конфигурируемое через JSON.
   - Вывод всех настраиваемых параметров и обработка ошибок.

2. **Этап 2 – Сбор данных**

   - Получение информации о прямых зависимостях пакета.
   - Поддержка тестового репозитория в формате Maven POM.
   - Вывод всех прямых зависимостей на экран.

3. **Этап 3 – Основные операции**

   - Построение графа зависимостей с учётом транзитивности с использованием BFS.
   - Учёт максимальной глубины анализа и фильтрации по подстроке.
   - Обработка циклических зависимостей.

4. **Этап 4 – Дополнительные операции**

   - Поддержка вывода порядка загрузки зависимостей (topological sort).
   - Обнаружение циклов и сравнение с результатами реального менеджера пакетов.

5. **Этап 5 – Визуализация**
   - Генерация графа зависимостей в формате Mermaid.
   - Сохранение изображения графа в формате PNG.
   - Демонстрация работы для нескольких пакетов.
